# Football Manager Application
## Overview

A GUI-based football management system built in Python that supports authentication, dynamic player signing and unsigning, team swapping, and active squad allocation. 
The application emphasizes validation logic, structured design, and real-time interface updates.

# Features
- Login authentication with error handling
- Player signing/unsigning with validation checks
- Team swapping with live dashboard updates
- Active squad allocation with swap logic
- Dynamic roster updates and state-based UI controls

# Tech Stack
Python • Tkinter • Object-Oriented Programming

# How to Run: 
- git clone https://github.com/shivang0127/football-manager.git
- cd football-manager
- python LoginView.py
